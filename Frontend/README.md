# RepLab Frontend

Mobile-first Vite + React + TypeScript app for the RepLab hackathon MVP.

## Local Development

From this `Frontend` directory, run:

```powershell
npm.cmd run dev
```

Use `npm.cmd` on Windows PowerShell so the command runs through the Node executable instead of the blocked `npm.ps1` script.

## Verification

```powershell
npm.cmd run build
npm.cmd run lint
```
